# Netflix Movies and TV Shows Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover trends, patterns, and insights about Netflix content.

## Objectives
- Analyze the distribution of Movies and TV Shows on Netflix.
- Explore trends in content released over the years.
- Identify the most common genres and ratings.
- Examine country-wise content distribution.
- Visualize findings using charts and graphs.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
The project uses the Netflix Movies and TV Shows dataset containing information such as:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre

## Analysis Performed
- Data Cleaning and Preprocessing
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Content Distribution Analysis
- Release Trend Analysis
- Genre and Rating Analysis
- Data Visualization

## Key Insights
- Compared the proportion of Movies and TV Shows available on Netflix.
- Identified trends in Netflix content releases over time.
- Explored the most frequent genres and audience ratings.
- Analyzed country-wise distribution of Netflix content.
- Presented findings through visualizations for better understanding.
